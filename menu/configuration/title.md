# Title

## Usage

```yaml
#
# Multiple titles
# Support color, functions, placeholder
#
Title:
  - 'Hello, TrMenu!'
  - 'Hello, %player_name%!'
  - 'Support Animated Titles'

#
# The update period (in ticks)
#
Title-Update: 40
```

```yaml
#
# Single static title
#
Title: 'TrMenu Menu'

#
# No need for this option
#
# Title-Update: -1
```
